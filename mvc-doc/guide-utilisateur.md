# Guide Utilisateur

Ce guide explique comment utiliser CSB Blog en tant qu'utilisateur final.

## Authentification

### Page d'accueil

À votre première visite, vous arrivez sur la page d'accueil qui affiche :

- Une liste des articles publiés
- Les articles les plus récents en premier
- Un aperçu du contenu de chaque article

### Inscription

Pour créer un compte :

1. Cliquez sur le lien "Inscription" en haut de la page
2. Remplissez le formulaire d'inscription :
   - **Nom d'utilisateur**: Votre identifiant unique
   - **Email**: Votre adresse email (utilisée pour la connexion)
   - **Mot de passe**: Un mot de passe sécurisé
3. Cliquez sur "S'inscrire"
4. Si la création réussit, vous êtes redirigé vers la page de connexion

**Conseils pour un mot de passe sûr:**
- Au moins 8 caractères
- Mélanger majuscules, minuscules, chiffres et symboles
- Éviter les mots courants ou informations personnelles

### Connexion

Pour vous connecter :

1. Cliquez sur "Connexion" en haut de la page
2. Entrez votre email et votre mot de passe
3. Cliquez sur "Se connecter"
4. Vous êtes redirigé vers la page d'accueil une fois connecté

### Déconnexion

Pour vous déconnecter, cliquez sur le lien "Déconnexion" (visible une fois connecté).

## Navigation principa

Une fois connecté, vous avez accès à :

- **Accueil**: Page d'accueil avec tous les articles
- **Recherche**: Rechercher des articles par titre ou contenu
- **Profil**: Voir et modifier vos informations de compte
- **Déconnexion**: Quitter votre session

## Consultation des articles

### Lister les articles

La page d'accueil affiche une liste paginée des articles. Pour naviguer :

- Les articles sont triés par date de publication (plus récents d'abord)
- Utilisez les contrôles de pagination pour voir plus d'articles
- Chaque article affiche son titre et une aperçu du contenu

### Lire un article complet

1. Cliquez sur le titre d'un article
2. Vous accédez au contenu complet de l'article
3. Les commentaires sont affichés sous l'article

## Gestion des commentaires

### Ajouter un commentaire

Pour ajouter un commentaire à un article :

1. Lisez l'article (accédez à sa page)
2. Trouvez la section de commentaires en bas de la page
3. Connectez-vous si vous ne l'êtes pas déjà
4. Remplissez le formulaire de commentaire
5. Cliquez sur "Publier le commentaire"
6. Votre commentaire apparait immédiatement

### Modifier un commentaire

Vous ne pouvez modifier que vos propres commentaires :

1. Trouvez votre commentaire sur la page de l'article
2. Cherchez le lien "Modifier" ou "Editer"
3. Modifiez le texte du commentaire
4. Cliquez sur "Mettre à jour"

### Supprimer un commentaire

Vous ne pouvez supprimer que vos propres commentaires :

1. Trouvez votre commentaire sur la page de l'article
2. Cherchez le lien "Supprimer"
3. Confirmez la suppression si demandé
4. Le commentaire est supprimé immédiatement

## Recherche d'articles

Pour rechercher des articles :

1. Cliquez sur "Recherche" dans le menu
2. Entrez des mots-clés (titre ou contenu)
3. Cliquez sur "Rechercher"
4. Les résultats correspondants sont affichés

**Conseils pour une meilleure recherche:**
- Utilisez des mots-clés spécifiques
- Essayez différentes variantes de vos recherches
- Les résultats incluent les correspondances partielles

## Gestion du profil

### Accéder à votre profil

1. Connectez-vous
2. Cliquez sur "Profil" dans le menu
3. Vous voyez vos informations actuelles

### Modifier vos informations

Vous pouvez modifier :

- **Nom d'utilisateur**: Votre identifiant public
- **Email**: Pour les notifications ou récupération de compte

Pour modifier :

1. Allez à votre profil
2. Cliquez sur "Modifier les informations"
3. Changez les champs désiré
4. Cliquez sur "Mettre à jour"

### Changer votre mot de passe

1. Allez à votre profil
2. Cliquez sur "Changer le mot de passe"
3. Entrez votre mot de passe actuel
4. Entrez le nouveau mot de passe deux fois
5. Cliquez sur "Mettre à jour le mot de passe"

## Tableau de bord administrateur

Si vous êtes un administrateur, vous avez accès à un tableau de bord spécial :

1. Cliquez sur "Admin" dans le menu (visible uniquement aux admins)
2. Vous accédez au tableau de bord administrateur

### Fonctionnalités administrateur

En tant qu'administrateur, vous pouvez :

- **Gérer les articles**
  - Créer de nouveaux articles
  - Modifier les articles existants
  - Supprimer des articles
  
- **Gérer les commentaires**
  - Afficher tous les commentaires du site
  - Modifier les commentaires des autres
  - Supprimer les commentaires inappropriés
  
- **Gérer les utilisateurs**
  - Afficher la liste de tous les utilisateurs
  - Consulter les informations des utilisateurs
  - Accorder ou retirer les droits d'administrateur

- **Consulter les logs**
  - Voir l'historique de toutes les actions
  - Consulter les tentatives de connexion
  - Suivre les modifications du contenu

## Conseils et bonnes pratiques

### Sécurité

- Choisissez un mot de passe fort
- Ne partagez pas votre mot de passe
- Déconnectez-vous après chaque session
- Vérifiez que l'URL est correctement formatée avant d'accéder

### Contenu

- Lisez les commentaires existants avant d'en ajouter
- Soyez respectueux dans vos commentaires
- Évitez le spam ou les messages hors-sujet
- Reportez les commentaires inappropriés aux administrateurs

### Navigation

- Utilisez la recherche pour trouver des articles spécifiques
- Vérifiez qui sont les contributeurs via le profil
- Consultez les articles les plus récents pour rester à jour

## Dépannage

### Je n'arrive pas à me connecter

- Vérifiez que vous utilisez le bon email et mot de passe
- Assurez-vous que les majuscules/minuscules sont correctes
- Si vous avez oublié votre mot de passe, contactez un administrateur
- Vérifiez que votre compte a bien été créé lors de l'inscription

### Mes modifications ne sauvegardent pas

- Vérifiez que vous avez attendu le message de confirmation
- Actualisez la page pour voir les modifications
- Vérifiez que votre connexion est toujours active
- Contactez le support si le problème persiste

### Je ne vois pas mes commentaires

- Vérifiez que vous êtes connecté
- Actualisez la page de l'article
- Les commentaires peuvent nécessiter une modération avant publication
- Contactez un administrateur si nécessaire

### Erreur lors de la publication d'un commentaire

- Assurez-vous que le champ de commentaire n'est pas vide
- Vérifiez que vous êtes connecté
- Le serveur peut être temporairement indisponible
- Réessayez quelques instants plus tard

## Accès à l'aide

Pour obtenir de l'aide supplémentaire :

- Consultez cette documentation complète
- Contactez l'administrateur du site
- Vérifiez les messages d'erreur affichés à l'écran
- Consultez les fichiers de log (pour administrative)
